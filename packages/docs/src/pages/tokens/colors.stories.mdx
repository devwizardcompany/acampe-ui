# Colors

Essas cores são parte do incrível design system da Acampe, uma empresa de software conhecida por sua criatividade e qualidade. Elas são usadas em diversos projetos, desde interfaces de usuário até aplicativos móveis e web. Essas cores cuidadosamente selecionadas trazem elegância e consistência visual para garantir uma experiência atraente para os usuários. Cada cor transmite uma sensação única, seja energia, confiança, renovação ou outras emoções. Com a paleta de cores Acampe UI, seus projetos ganharão um visual sofisticado e harmonioso, deixando sua marca no mundo digital.
